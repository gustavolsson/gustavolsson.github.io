Sailor (Ludum Dare #11)

The game let’s you build a boat and sail the sea. Try to stay afloat as long as possible, it’s not as easy as it sounds!

Ludum Dare is a 48h (and nowadays also 72h) game programming competition. The quality of my code written for these competitions is horrendous, but the focus is on completing a big undertaking in a very limited amount of time.
